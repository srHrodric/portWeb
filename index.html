Serviços

Atuo na elaboração de projetos para Web, como sites ou sistemas. Sendo responsável pela criação de toda parte visual do site e funcionalidades de um sistema. No aspecto visual, desenvolvo toda à 'interface' e à usabilidade do site, tratando das partes visuais ou de interação quando se está em um site. Já em projetos de sistemas para Web, desenvolvo todas as funcionalidades para o seu funcionamento, utilizando de linguagens como PHP e JavaScript, por exemplo.

Requisitos

Levantar requisitos para sincronizar a expectativa do cliente com o produto final. Devendo atender a expectativa do cliente, o máximo possível.Reduzindo custos e potencial retrabalho.

Funcionalidades

Definir e ajustar como as funções do site para que tenha um melhor desempenho em termos de design, navegação, acessibilidade, adaptação para dispositivos móveis, entre outros.

Design

Elaboração do projeto estético e funcional de um site, para que possa ajustar as funções para qual foi desenvolvido com layout do site, integrando sua funcionalidade com o designer.

Elaboração
